Open Large Language Models (LLMs) Marketing Analysis

  1. Eugeneyan. (2023).  open-llms  [Survey]. GitHub.  https://github.com/eugeneyan/open-llms

This scatter plot effectively visualizes the relationship between the release dates and the number of parameters in various large language models (LLMs).

By utilizing a linear scale for the x-axis ("LLM's Release Date") and a square root scale for the y-axis ("LLM's Parameters in Billion"), the plot offers a clear depiction of how LLMs have evolved over time.

It reveals that the top LLMs' number of parameters grows following a square root ratio, indicating rapid but progressively stabilizing expansion in model sizes.

Additionally, it highlights a trend where more recent LLMs are increasingly open and expansive, reflecting a shift toward more accessible and larger-scale models in the field.
